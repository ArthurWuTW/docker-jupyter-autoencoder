    ~~Issue: [https://github.com/moby/moby/issues/48068](https://github.com/moby/moby/issues/48068)~~  
    文件註記：需要測試 multiple manager 情境  
    - 參考：[Swarm Admin Guide](https://docs.docker.com/engine/swarm/admin_guide/)  
    - Manager 狀態說明：`Reachable` 代表節點是 manager node，參與 Raft 共識，如果 leader node 掛掉，它有資格被選為新的 leader。  
    - 參考：[Manage Nodes](https://docs.docker.com/engine/swarm/manage-nodes/)  
    - 參考：Docker Swarm manager 範例  
      ```
      docker node promote node01
      docker node promote node02
      ```
      - 測試結果示例：
        ```bash
        docker node ls
        # stkyf5b9u0rwqkz49b9bbur9l *   swarmmanager   Ready     Active   Leader
        # tp121f85etiq2eg9dov93hxrj     swarmnode1     Ready     Active
        # ewblpdtfso1zi9kv24on9m6xb     swarmworker2   Down      Active
        docker service ls
        # zvdkky1hbdy9   helloworld   replicated   7/5   alpine:latest
        docker service ps helloworld
        # ico9n4fwe003   helloworld.1   alpine:latest   swarmnode1   Running
        # mt1b8tcfvol8   _ helloworld.1 alpine:latest   swarmworker2 Shutdown
        ```
