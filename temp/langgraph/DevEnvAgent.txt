- Run Moby Debug Mode in VSCode
  1. 安裝 VSCode 插件：Dev Containers
  2. 重啟 VSCode，顯示 container bash
  3. VSCode 指令：Reopen vscontainer
  4. `launch.json` 設定
     ```json
     {
         "version": "0.2.0",
         "configurations": [
             {
                 "name": "Launch Package",
                 "type": "go",
                 "request": "launch",
                 "mode": "auto",
                 "program": "${fileDirname}"
             }
         ]
     }
     ```
  5. 可能錯誤：
     - error: Invalid userlane-proxy-path: userland-proxy is enabled, but userland-proxy-path is not set
     - Running modprobe bridge br_netfilter failed with message: " error="exec: \"modprobe\": executable file not found in $PATH
       → iptables 相關
  6. `/etc/docker/daemon.json` 範例：
     ```json
     {
         "runtimes": {
             "crun": {
                 "path": "/usr/local/bin/crun"
             }
         },
         "features": {
             "containerd-snapshotter": false
         }
         "userland-proxy": false,
         "iptables": false
     }
     ```
  7. reinstall iptables
     ```bash
     apt remove iptables
     apt install iptables
     ```
  8. run `docker.go` in debug mode

- Run Moby inside Dev Container
  - [reference](https://github.com/moby/moby/blob/master/docs/contributing/set-up-dev-env.md)
  1. Host machine VM 安裝 docker
  2. Host machine VM clone 專案、建立 container  
     ```bash
     git clone https://github.com/moby/moby
     cd moby
     git checkout -b dry-run-test
     make BIND_DIR=. shell
     ```
  3. dev container reinstall iptables，再跑 dockerd  
     ```bash
     apt remove iptables
     apt install iptables
     hack/make.sh binary
     make install
     dockerd -D
     ```
  4. host 端測試  
     ```bash
     docker run hello-world
     # Hello from Docker!
     ```

- Run Moby Project and Create Docker Swarm Manager in VSCode
  1. 在 Rocky Linux 8 VM 停用 firewalld
     ```bash
     systemctl stop firewalld
     systemctl disable firewalld
     ```
  2. VSCode: 使用 devcontainer 跑 Moby
  3. VSCode: 設定 Local Port Host → listen on 0.0.0.0
  4. VSCode: Run Moby Project in debug mode，執行 `docker swarm init ...`
  5. VM Moby: `netstat -anp | grep :2377`
  6. Rocky Linux 8 VM: `docker swarm join ...`
  7. VSCode: 確認 join 結果
