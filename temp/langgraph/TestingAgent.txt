### Run All Integration Test
1. 建立 container，執行 dockerd
2. 修改 hack/make.sh，新增：
   apt remove iptables -y
   apt install iptables -y
3. 執行：
   make test-integration
- 文件附有測試畫面截圖 (image.png, scaled-1680-/JuBimage.png)

---

### Run Single Unit Test in Debug Mode
1. 在 Debug 模式下執行 daemon.go
2. 在 bridge_test.go 點擊綠色 > 按鈕執行測試
3. 測試結果會顯示在 TEST RESULT 面板
- 文件附有截圖 (image.png, scaled-1680-/9m2image.png 與 yszimage.png)

---

### Run Single Integration Test
1. 修改 hack/make.sh，同樣新增：
   apt remove iptables -y
   apt install iptables -y
2. 在 Host 設定環境變數：
   export TESTFLAGS='-test.run TestBridgeICC/IPv4_non-internal_network'
   export TEST_INTEGRATION_DIR=./integration/networking
3. 執行：
   make test-integration
- 文件附有測試截圖 (image.png, scaled-1680-/gdnimage.png)
